Purpose of Nginx:


The purpose of Nginx is both security and stability for the client side.
Nginx provide us with a reverse proxy which helps us in protecting our self from for instance DoS and DDoS
Nginx can also assist the client side due to cache thereby reducing the load of the client.
We also utilize it to get a SSL certification since it is compatiable.

Its to add security between the client and the server. The server sends the certificate to the client who then checks if it's valid
This is done by checking the name of the certificate vs the servers name.
